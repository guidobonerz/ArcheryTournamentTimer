# Archery Tournament Timer

Functions Keys
- [T] Tournament View/Turnier Ansicht
- [H] Help View/Hilfe Ansicht
- [ESC] Interrupt current yoke/Passe vorzeitig beenden
- [N] Restart/Neustart
- [S] Soundcheck
- [F11] Fullscreen/Vollbild
- [\X] Exit program/ Programm beenden
